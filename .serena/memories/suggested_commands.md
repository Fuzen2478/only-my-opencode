**Key Commands:**

*   `bun run typecheck`: Type check the project.
*   `bun run build`: Build ESM + declarations + schema.
*   `bun run rebuild`: Clean + Build.
*   `bun test`: Run all test files.
*   `bun run build:schema`: Builds the configuration schema.
*   `bun run build:binaries`: Builds platform-specific binaries.
*   `bun run clean`: Removes the `dist` directory.
*   `bun run postinstall`: Runs post-installation scripts.
*   `bun run prepublishOnly`: Cleans and builds before publishing.
