The project is primarily built with TypeScript and uses Bun as the package manager. Key dependencies include:
*   @ast-grep/cli, @ast-grep/napi: For AST-aware code searching and rewriting.
*   @fuzen2478/comment-checker: A custom comment checking utility.
*   @opencode-ai/plugin, @opencode-ai/sdk: Core OpenCode functionalities.
*   zod: For schema validation.
*   bun-types: For TypeScript type definitions.
